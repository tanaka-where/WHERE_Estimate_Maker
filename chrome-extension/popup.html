WHERE 見積書メーカー — 設定

エディタ URL

GitHub Pages 等にホスティングした URL を指定。空の場合は manifest 内のデフォルト URL が使用されます。

Google Drive

Drive のフォルダURLの末尾の文字列。manifest.json の oauth2.client_id 設定が必要です。

Slack

投稿先チャンネルは Webhook 作成時に設定します。

ヘルプ

  • 使い方: Gmail でメール作成 → ツールバーの「WHERE 見積書を作成」ボタン
  • 算出ロジック: 見積書メーカー右上の「算出ロジック」リンク
  • 変数の修正: GitHub の variables.json を PR で更新
