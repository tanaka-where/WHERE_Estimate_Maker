# WHERE 見積書メーカー — 利用開始ガイド

営業メンバー向けの導入手順です。**初回セットアップは10分程度**で完了します。

---

## できること

- Gmail のメール作成画面から **「WHERE 見積書を作成」ボタン** で起動
- 必須項目を入力するだけで見積書を生成
- **PDF を自動でメールに添付**
- **Google Drive にも自動保存**（保存先フォルダは毎回選べます）
- 算出ロジック（プラン単価・税率・係数）はWeb上でいつでも確認可能

ファイル名は自動で：`【顧客名御中】WHERE利用費見積書_YYYYMMDD.pdf`

---

## インストール手順（初回のみ）

### Step 1: 拡張機能の ZIP をダウンロード

管理者から配布される `WHERE_Estimate_Maker_extension.zip` をデスクトップなど任意の場所に保存してください。

### Step 2: ZIP を解凍

ZIP ファイルを **右クリック → 「すべて展開」**（Mac の場合はダブルクリックで自動解凍）

解凍後、`chrome-extension` フォルダが出来ます。**このフォルダは消さずに残してください**（拡張機能はこのフォルダを参照し続けます）。

> 推奨配置場所：`ドキュメント\WHERE_Estimate_Maker\chrome-extension`

### Step 3: Chrome に拡張機能を読み込む

1. Chrome のアドレスバーに以下を入力して Enter
   ```
   chrome://extensions/
   ```

2. 画面右上の **「デベロッパーモード」** スイッチを **ON** にする

3. 左上に表示される **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリック

4. Step 2 で解凍した **`chrome-extension` フォルダ**（中に `manifest.json` がある階層）を選択 →「フォルダーの選択」

5. 拡張機能カードが表示され、**WHERE 見積書メーカー — Gmail 連携** が有効化されればOK

### Step 4: 拡張機能のピン留め（任意・推奨）

Chrome ツールバー右上の **拡張機能アイコン（パズルピース🧩）** をクリック →
**WHERE 見積書メーカー** の右側にあるピンマークをクリックしてピン留め。

これでツールバーから常にアクセスできます。

## 使い方

### 基本フロー

1. **Gmail を開く**（既に開いていたら **F5 キーで再読み込み**）
2. **「作成」** ボタンで新規メール画面を開く
3. 送信ボタンの右隣に **紺色の「WHERE 見積書を作成」ボタン** が表示されるのでクリック
4. 別ウィンドウで **見積書メーカー** が起動
5. **起動モーダル** に必須項目を入力：
   - 顧客名（必須）
   - 件名・営業担当・契約期間・プラン・各日付
6. **「編集をスタート」** をクリック
7. サイドバーで詳細を編集（DM・反響対応等のオプション、カスタム項目）
8. プレビューで内容を確認
9. 右上の **「完了 → メールに添付」** ボタンをクリック
10. **Drive 保存先選択モーダル** が表示される
    - フォルダの URL（または ID）を貼り付け
    - 履歴から選択も可能
    - 空欄なら「マイドライブ」直下に保存
    - 「Drive 保存をスキップ」も可能
11. 「この場所で確定」をクリック
12. PDF がメールに自動添付され、Drive にも保存される

### 初回のみ：Google 認証

最初の Drive 保存時に **Google の同意画面** が表示されます：

1. 「**WHERE 見積書メーカーが Google アカウントにアクセスしようとしています**」と表示
2. 「**続行**」または「**許可**」をクリック
3. アクセス権限「**Google ドライブのファイルを表示・管理**」を許可

これで2回目以降は自動的に保存されます。

### Drive 保存先フォルダの指定方法

1. **Google Drive を開く**：ブラウザで https://drive.google.com/drive/my-drive
2. 保存したいフォルダを **ダブルクリックで開く**
3. ブラウザの **URL バーをコピー**
   - 例: `https://drive.google.com/drive/folders/1AbC2DefGhI3jKlMno...`
4. 見積書メーカーの **Drive 保存先選択モーダル** の入力欄に **貼り付け**
5. 「表示名（任意）」に分かりやすい名前を付けると履歴から選びやすい
   - 例: `2026年Q2_提案書`、`顧客A`

一度入力したフォルダは **履歴から1クリックで再選択可能** です（最大8件保存）。

## トラブルシューティング

### Gmail にボタンが出ない
1. 拡張機能が有効になっているか確認（`chrome://extensions/`）
2. Gmail を **F5 で再読み込み**
3. 新規メール作成画面を開く

### 添付されない・エラーが出る
1. 拡張機能の **更新ボタン（↻）** で再読み込み
2. Chrome を再起動
3. それでもダメなら管理者へ連絡（F12 で開いた Console のログがあると早い）

### 「Google API has not been used」エラー
管理者側の設定問題なので連絡してください。

### Drive に保存されない
1. 初回 OAuth 同意でアクセス許可したか確認
2. 設定で「Drive 保存」が ON になっているか確認（拡張機能アイコン → ポップアップ）

### 拡張機能の自動更新

ZIP 配布版は **自動更新されません**。新バージョンが配布されたら：

1. 古い `chrome-extension` フォルダを **新しい ZIP の中身で上書き**
2. `chrome://extensions/` で WHERE 見積書メーカーカードの **更新ボタン（↻）** をクリック

## 設定変更（任意）

拡張機能アイコン → **WHERE 見積書メーカー** をクリックでポップアップが開きます：

| 項目 | 説明 |
|------|------|
| エディタ URL | 通常空欄でOK（デフォルトURLが使用される） |
| Drive 連携 | OFF にすると Drive 保存をスキップ |
| 保存先フォルダ ID | デフォルトの保存先（編集時のモーダルで都度上書き可能） |
| Slack 連携 | 現在未設定（必要なら管理者へ連絡） |
